## Getting Started

Install all dependencies:

```bash
yarn
```

## Building the project

Compile all contracts.

```bash
yarn build
```

## Deploying Contracts

Deploy contracts on any EVM chain:

```bash
yarn deploy
```

## Releasing Contracts

Publish contracts on Thirdweb. Will be pinned on IPFS:

```bash
yarn publish
```
